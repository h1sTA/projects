# Telegram Weather Bot

Телеграм-бот, который отправляет текущую погоду по названию города.

## Функции:
- Получение погоды по городу
- Оповещение каждый день в указанное время

## Как запустить?
1. Склонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Вставить свой токен в `config.py`
4. Запустить: `python bot.py`
